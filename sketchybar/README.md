# Sketchybar Configuration

## 📁 Project Structure

```
~/.config/sketchybar/
├── README.md                    # This documentation file
├── sketchybarrc                # Main configuration file
├── label.drawing=off           # Configuration flag file
└── plugins/                    # Custom plugin scripts
    ├── audio_listener.applescript  # Audio listener AppleScript
    ├── audio_listener.sh          # Audio listener shell script
    ├── audio_trigger.sh           # Audio trigger script
    ├── audio.sh                   # Audio volume display
    ├── battery.sh                 # Battery status and charging
    ├── ble_headset.sh             # Bluetooth headset detection
    ├── charging.sh                # Charging status indicator
    ├── clock.sh                   # Date and time display
    ├── front_app.sh               # Active application display
    ├── network.sh                 # Network bandwidth monitor
    ├── power.sh                   # Power management
    ├── space.sh                   # Mission Control spaces
    └── volume.sh                  # Volume control
```
